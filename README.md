# 🚀 AcademiaDoProgramadorBackEnd

Repositório destinado aos projetos desenvolvidos durante a **Academia do Programador 2026**, com foco em **lógica de programação**, **desenvolvimento backend** e **boas práticas de código**.

O objetivo deste repositório é acompanhar a evolução técnica ao longo do curso, reunindo exercícios, aplicações de console e projetos práticos.

---

## 📚 Projetos

### 🧮 Calculadora Console

Aplicação de console que realiza operações matemáticas básicas e mantém histórico das operações realizadas.

🔗 Acesse o projeto:
➡️ `CalculadoraV2/`

**Funcionalidades:**

* Soma
* Subtração
* Multiplicação
* Divisão
* Tabuada
* Potência
* Raiz Quadrada
* Histórico de Operações
