# 🧮 Calculadora Console

## 📌 Introdução

Projeto desenvolvido durante a **Academia do Programador 2026** com o objetivo de revisar conceitos fundamentais de **lógica de programação** e prática com **C# em aplicações Console**.

O sistema simula uma calculadora interativa executada via terminal, permitindo ao usuário realizar diferentes operações matemáticas e visualizar o histórico de cálculos realizados.

---

## 🚀 Funcionalidades

* ➕ Soma
* ➖ Subtração
* ✖️ Multiplicação
* ➗ Divisão
* 🔢 Tabuada
* 🧠 Potência
* 📐 Raiz Quadrada
* 📝 Histórico de Operações

---

## 🛠 Tecnologias Utilizadas

* C#
* .NET
* Console Application

---

## ▶️ Como executar o projeto

1. Clonar o repositório:

```
git clone https://github.com/Vanderluizsj/AcademiaDoProgramadorBackEnd.git
```

2. Acessar a pasta do projeto:

```
cd CalculadoraV2
```

3. Executar o projeto:

```
dotnet run
```
